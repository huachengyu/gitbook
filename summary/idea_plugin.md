### IDEA Plugin 备份
@Date 2017.05.24

> key promoter
 * 在操作IDEA功能时会提示对应的快捷键
 
> maven helper
 * 分析maven依赖
 
> lombok
 * 支持编译logbok注解

> FindBugs
 * 扫描工程发现代码明显错误告警问题

> Alibaba Java Coding Guidelines
 * 阿里巴巴代码规范

> Grep Console
 * IDEA 控制台日志输出,颜色区分
 
> GsonFormat
 * Json格式化
 
> Rainbow Brackets
 * 代码区块, 不同颜色区分
